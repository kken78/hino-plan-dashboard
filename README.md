# 第6次日野町総合計画 進捗ダッシュボード（プロトタイプ）

滋賀県日野町の「第6次日野町総合計画」に関する3つの公開資料を統合し、計画の進捗を視覚的に把握できるよう構築した、オープンデータ利活用のための実証実験プロジェクトです。

外部ビルド不要の SPA（シングルページアプリケーション）で、`index.html` を開くだけで動作します。デジタル庁「ダッシュボードデザインガイドブック」の考え方を参考に、認知負荷の低い UI を意識しています。

## 特徴

**1. 3つの情報源を「34分野」で横断**
総合計画の構造（5つの政策の柱 → 10政策 → 34分野）を共通の軸として、以下を1つのダッシュボードに統合しています。

- **基本計画の成果指標（KPI）** … 34分野中30分野・計76指標。基準値 → 中間目標(R7) → 最終目標(R12) の3点を収録
- **実施計画（R7-R9）** … 分野別の概算事業費（3年計・約186億円）と主要事業
- **住民意識調査（R6実施）** … 満足度・協働・幸福度・情報入手・属性など（有効回答1,206件）

**2. 4つのタブ構成**

| タブ | 内容 |
| --- | --- |
| 概要 | 計画の現在地（5年目・中間年）、KPI数、予算規模、回収率、幸福度、柱別予算配分 |
| KPI・進捗 | 分野別の成果指標。基準→最終目標を100%とし中間目標の位置をバー表示。改善の向き（増加で改善/減少で改善/中立）を色分け |
| 実施計画 | 分野別の事業費ランキング、主要事業 上位12 |
| 住民の声 | 意識調査の設問別グラフ・回答者属性 |

**3. 改善の「向き」を明示**
指標により「増えると良い」「減ると良い」「維持・中立」が異なるため、緑（増加で改善）・赤（減少で改善）・灰（中立）で色分けし、文章でも明記しています。

**4. モバイル・アクセシビリティ対応**
スマートフォンでは1列レイアウト。キーボードフォーカス表示、`prefers-reduced-motion`（視差効果を減らす設定）にも対応しています。

## ファイル構成

```
index.html        … 本体（構造とスタイル）
README.md         … 本ファイル
ogp.png           … SNS共有時のサムネイル画像（1200×630px）
assets/
  app.js          … 描画ロジック・タブ制御・達成度計算
  data_bundle.js  … data/ のJSONを結合した埋め込みデータ（index.htmlはこれを読み込む）
  ogp.png         … OGP画像（同上）
  ogp.svg         … OGP画像の編集元（差し替え用）
data/
  kpi.json        … 全34分野の成果指標
  plan.json       … 実施計画（分野別予算・主要事業）
  survey.json     … 住民意識調査
build.py          … data/*.json から assets/data_bundle.js を再生成するスクリプト
```

データを更新する場合は `data/*.json` を編集し、`python3 build.py` で `assets/data_bundle.js` を再生成してください。

## OGP（SNS共有時のサムネ）の設定

`ogp.png`（1200×630px）を同梱しています。GitHub Pages で公開する際は、`index.html` の `<head>` 内の `og:image` / `twitter:image` を公開URLの絶対パスに書き換えると、SNSやチャットで確実にカード画像が表示されます。

```html
<meta property="og:image" content="https://（ユーザー名）.github.io/（リポジトリ名）/ogp.png">
<meta name="twitter:image" content="https://（ユーザー名）.github.io/（リポジトリ名）/ogp.png">
```

画像を差し替えたい場合は `assets/ogp.svg` を編集して PNG に書き出してください。公開後は Facebook Sharing Debugger や X Card Validator でキャッシュを更新すると確実に反映されます。

## ローカルでの確認

`index.html` をブラウザで直接開くだけで動作します（データはJSに埋め込み済みのため簡易サーバ不要）。

## 今後の拡張余地

- **KPIに「現在地（実績値）」を重ねる**：現在は基準値→目標値のみ。各指標の最新実績が公表されれば、達成率の実進捗を表示できます。
- **住民意識調査の「定着度×優先度」マトリクス**：34施策を4タイプに分類する散布図は、概要版PDFでは画像化されており数値が読み取れませんでした。元の詳細報告書（数表）が入手できれば、最優先改善領域を可視化する目玉ビューを追加できます。
- **実施計画の経年比較**：過去年度（R3-R5〜R6-R8）のPDFが公開済みのため、事業の追加・変更の経年比較が可能です。

## 注意事項

- 本リポジトリのデータは公開PDFから独自に抽出して作成したものです。読み取り誤差や変換ミスを含む可能性があるため、正確な数値が必要な場合は必ず日野町公式の元資料（PDF）をご確認ください。
- 一部分野は中間目標の年次がR7と揃っていない（R5・R6表記）、基準値の年次が分野によりR1/R2見込み/H27〜H28と異なる、などの点があります。データには年次を併記しています。
- 本プロジェクトは日野町の公式システムではなく、オープンデータ推進に向けた実証実験（プロトタイプ）です。
